01 — Sobre el Proyecto

Sobre el Proyecto

Conoce el origen, motivación y contexto de esta aplicación desarrollada durante el ciclo formativo de DAM.

Introducción

El proyecto realizado tiene el propósito de ser utilizado para gestionar los videojuegos registrados en la aplicación guardándolos en listas personalizadas y dejando reseñas de estos. La gestión de estos videojuegos la decide el propio usuario. Pudiendo guardar videojuegos que desea comprar, que ya ha jugado, se ha terminado o de cualquier manera que se desee.

Nombre de la aplicación: EMA

Plataformas objetivo: Android / Windows / Web

Lenguaje: Java

Base de datos: MySQL e Hibernate

Metodología: Scrum

Origen y Contextualización

El origen de la idea de crear este servicio viene de la necesidad de gestionar con facilidad artículos deseados. Usando como referencia la conocida página web de JustWatch, se nos ocurrió replicarlo y enfocarlo al ámbito de videojuegos. También, para obtener más referencias, nos inspiramos en el funcionamiento de cómo en AlieExpress o Amazon se pueden crear las listas. De tal manera se decidió que la funcionalidad sería poder gestionar tus propias listas para organizar los videojuegos deseados. Ya sea separándolos por plataformas o por lo que el usuario guste.

🔍

Problema Detectado

Los jugadores suelen tener problemas para organizar sus videojuegos deseados, jugados o completados, y no existe una herramienta simple que centralice esta información de forma personalizada.

💡

Solución Propuesta

La aplicación organiza videojuegos en listas personalizadas por estado y plataforma, permitiendo al usuario llevar un control sencillo y centralizado de sus títulos.

🎯

Público Objetivo

Pensada para jugadores de todas las edades que deseen organizar y controlar sus videojuegos de forma sencilla y estructurada.
